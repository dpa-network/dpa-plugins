# DPA Plugins

**Open-source Claude Code plugins by [DPA (Decentralized Protection Alliance)](https://dpa.network).**

A collection of tools designed to bring more transparency to AI systems — for the benefit of both users and AI.

## Available Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| [context-monitor](https://github.com/dpa-network/claude-context-monitor) | Real-time context window awareness for AI | `claude plugin install context-monitor@dpa-plugins` |

## How to Use

### Add this marketplace

```bash
claude plugin marketplace add dpa-network/dpa-plugins
```

### Install a plugin

```bash
claude plugin install context-monitor@dpa-plugins
```

## About DPA

DPA (Decentralized Protection Alliance) is a collective focused on improving transparency and fairness in how AI systems interact with the people who use them. We build open-source tools, research AI company practices, and foster open dialogue about the future of AI.
